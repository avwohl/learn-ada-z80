# Variables and Constants

Data storage in Ada.

## Examples

| File | Concept |
|------|---------|
| [variables.adb](variables.adb) | Variable declarations |

## Key Concepts

```ada
-- Variables
Count : Integer := 0;
Name  : String(1..20);

-- Constants
Max_Size : constant := 100;
Pi       : constant := 3.14159;

-- Multiple declarations
X, Y, Z : Integer := 0;
```
